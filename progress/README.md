# 开发进度与 Plan 文档

本目录存放 Video Agent 项目的开发总结与各阶段 Plan 文档。

## 目录结构

```
progress/
├── README.md           # 本文件
├── PROGRESS.md        # 开发进度总结（后端/前端状态、设计要点）
├── Sprint6_交接说明.md # Sprint 6 交接说明
└── plans/             # 各阶段 Plan 文档
    ├── 视频生成agent系统_1d7d83ac.plan.md
    ├── n8n_风格节点编排器_0fb856c6.plan.md
    └── ... (其他 plan 文档)
```

## 使用说明

- `PROGRESS.md`：当前开发状态、待修复问题、设计要点
- `plans/`：各功能模块的详细 Plan，按开发阶段归档
