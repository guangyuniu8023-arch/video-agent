# 修改 image_generate_rule 输出格式为扁平数组

## 变更范围

文件：[/Users/bytedance/Downloads/## 角色定义.ini](/Users/bytedance/Downloads/## 角色定义.ini)

## 具体修改点

### 1. 输出格式定义（第83-95行）

将 `image_generate_rule` 的格式描述从字典改为扁平数组：

**当前格式：**

```json
"image_generate_rule": {
  "图片": [...],
  "IP": [...],
  "ID": [...]
}
```

**目标格式：**

```json
"image_generate_rule": [...]  // 按分镜顺序，直接包含 tool 对象
```

### 2. 更新全部 10 个 Few-shot 示例的输出

每个示例都需要去掉 "图片"/"IP"/"ID" 的包裹层，直接将 tool 对象放入扁平数组：

- **示例1**（第281-289行）：`"图片": [{...}]` -> `[{...}]`
- **示例2**（第313-319行）：`"图片": [{...}]` -> `[{...}]`
- **示例3**（第343-351行）：`"图片": [{...}]` -> `[{...}]`

- **示例5**（第406-413行）：`"图片": [{...}]` -> `[{...}]`
- **示例6**（第436-442行）：`"IP": [{...}]` -> `[{...}]`
- **示例7**（第464-471行）：`"ID": [{...}]` -> `[{...}]`
- **示例8**（第493-500行）：`"IP": [{...}], "ID": [{...}]` -> `[{...}, {...}]`（混合主体合并为一个扁平数组）
- **示例9**（第526-532行）：`"图片": [{...}]` -> `[{...}]`
- **示例10**（第553-563行）：`"图片": [{...}, {...}]` -> `[{...}, {...}]`
