# 剧本规范化处理方案

## 输入解析

- **图片数量**：1张（AI形象）
- **图片1**：[AI形象] → 对应槽位 `{role_0}`
- **匹配规则**：单图上传 + 无显式引用（文本中无"图1"、"图一"等引用）

## 适用规则

根据核心规则第3条 **隐式引用补全**：

> 单图上传 + 无显式引用：找到文本中指代人物的词（人物描述词如"一个女生"、人称代词如"我"等），将其**替换**为 `{role_0}`。

## 需要替换的内容


| 原文        | 替换为        | 说明          |
| --------- | ---------- | ----------- |
| `一个女生博主`  | `{role_0}` | 人物描述词       |
| `我`（出现3次） | `{role_0}` | 人称代词，指代同一人物 |


**注意**：文本中的"你"是面向观众的呼唤（"你看"、"快去试试吧"），不是指代图片中的人物，因此不替换。
